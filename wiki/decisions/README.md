# Decisions

ADR-style records written when a riff graduates into a committed direction, or when a contested technical choice is resolved. Each entry includes:

- **Context** — what's the situation
- **Decision** — what we chose
- **Consequences** — what this closes off or enables
- **Alternatives considered** — the paths not taken and why

Dated. Not modified after recording — if the decision reverses later, a new ADR supersedes it.

## Format

File name: `NNN-short-slug.md`. Number sequentially from 001.
